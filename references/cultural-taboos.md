# 文化禁忌速查手册

## 一、绝对不能送的物品

| 禁忌物品 | 原因 | 谐音/象征 |
|---------|------|----------|
| **钟（时钟）** | "送钟"谐音"送终"，暗示对方死亡 | 送终 |
| **伞** | "伞"谐音"散"，暗示分离、散伙 | 散 |
| **鞋** | "鞋"谐音"邪"，且有"送人走"之意 | 邪/送人走 |
| **梨** | "梨"谐音"离"，暗示分离 | 离 |
| **绿帽子** | 暗示配偶出轨，中国文化中最严重的侮辱之一 | 被绿 |
| **刀/剪刀/利器** | 暗示"一刀两断"，切断关系 | 一刀两断 |
| **手帕/毛巾** | 丧事中常用于擦泪，关联悲伤 | 丧事用品 |
| **蜡烛** | 白色蜡烛用于丧葬祭祀 | 丧葬 |
| **菊花（白/黄）** | 丧葬用花，用于祭祀亡者 | 祭祀 |
| **白色花束** | 白色在中国文化中与死亡、丧事相关 | 丧事 |
| **药品/补品（给健康人）** | 暗示对方身体不好 | 咒人生病 |
| **鱼缸** | "鱼缸"谐音"愚刚"，且送鱼缸=送财运 | 送财运 |
| **空钱包** | 送空钱包=送走财运 | 送财运 |
| **枕头** | 有"同床共枕"之意，关系不亲密不宜送 | 亲密关系 |
| **布娃娃/人形玩偶** | 有人认为会招邪，尤其在风水中 | 招邪 |
| **来历不明的古董** | 可能带有前主人的气场/邪气 | 带邪气 |
| **水仙花** | 有"水性杨花"之意 | 不忠 |
| **仙人掌/带刺植物** | 暗示对方"带刺"、难相处 | 带刺 |
| **书（某些场合）** | "书"谐音"输"，送赌徒/商人大忌 | 输 |
| **李子（送病人）** | "李"谐音"离"，送病人不吉利 | 离世 |
| **柿子（送病人）** | 部分地区认为不吉利 | — |

**特殊情况**：
- 如果一定要送钟，可以让对方给你 1 元钱，变成"买钟"而非"送钟"
- 如果一定要送伞，可以说"这是遮阳伞"避开谐音
- 如果一定要送鞋，可以让对方给你 1 元钱

## 二、数字禁忌

### 禁忌数字

| 禁忌 | 原因 | 说明 |
|------|------|------|
| **数字 4** | "四"谐音"死" | 最严重禁忌，避免4、14、24、34、40、44等 |
| **数字 13** | 受西方文化影响 | 部分年轻人忌讳 |
| **数字 250** | "二百五"指傻子、愚蠢的人 | 金额避免250 |
| **单数（喜事）** | 中国讲究"好事成双" | 喜事礼品数量宜双数 |
| **数字 3（送礼数量）** | "三"谐音"散" | 部分地区忌讳 |
| **数字 5（某些场合）** | "五"谐音"无" | 送礼金额避免 |

### 吉利数字

| 数字 | 寓意 | 适用场景 |
|------|------|---------|
| **168** | "一路发" | 通用 |
| **288** | "发" | 通用 |
| **366** | "顺" | 通用 |
| **520** | "我爱你" | 恋人/配偶 |
| **600** | "顺" | 通用 |
| **666** | "六六大顺" | 通用 |
| **800** | "发" | 通用 |
| **888** | "发发发" | 较高金额 |
| **999** | "长长久久" | 特殊单数例外 |
| **1314** | "一生一世" | 恋人/配偶 |
| **1666** | "一路顺" | 较高金额 |
| **1888** | "要发发发" | 高档 |

### 白事特殊数字

- 以**单数**为主：101、201、501
- 带 **1** 结尾：象征"一生一世"缅怀
- **忌双数、忌4**

## 三、颜色禁忌

| 颜色 | 禁忌场景 | 原因 |
|------|---------|------|
| **白色** | 婚礼、生日、春节 | 丧葬主色，代表死亡 |
| **黑色** | 婚礼、春节 | 丧葬色，不吉利 |
| **绿色** | 送给男性 | "绿帽子"暗示出轨 |
| **黄色** | 丧葬场合 | 部分地区丧葬用色 |
| **纯白包装** | 任何喜庆场合 | 联想丧事 |
| **纯黑包装** | 喜庆场合 | 压抑、不吉利 |

**吉利颜色**：
- **红色** - 喜庆、好运、婚庆首选
- **金色** - 富贵、尊贵
- **粉色** - 温馨、浪漫
- **暖色系** - 一般安全

## 四、花卉禁忌

| 花卉 | 禁忌场景 | 原因 |
|------|---------|------|
| **白色菊花** | 任何场合（除祭祀） | 祭奠亡者专用 |
| **黄色菊花** | 任何场合（除祭祀） | 祭奠亡者专用 |
| **白色花束** | 婚礼、生日、探病 | 丧葬联想 |
| **杜鹃花** | 送人 | "杜鹃啼血"，悲伤 |
| **扶桑花** | 送人 | "扶桑"谐音"服丧" |
| **茉莉** | 送恋人 | "茉莉"谐音"没利"、"末离" |
| **水仙** | 送女性 | 暗示"水性杨花" |

**安全花卉**：
- **百合** - 百年好合（婚礼首选）
- **牡丹** - 富贵吉祥
- **兰花** - 高雅
- **向日葵** - 阳光、积极
- **红玫瑰** - 爱情（恋人之间）
- **康乃馨** - 感恩（送长辈/母亲）
- **剑兰/唐菖蒲** - 节节高升

## 五、地区性禁忌差异

| 地区 | 特殊禁忌 | 说明 |
|------|---------|------|
| **广东/香港** | "书"谐音"输"更忌讳；忌送钟、梨、伞 | 春节"利是"小额为主(5-20元) |
| **福建/闽南** | "伞"特别忌讳；忌送"四"的数量 | — |
| **上海** | 苹果送病人有争议（"苹"谐音"病"） | 份子钱较高，婚礼1000起 |
| **台湾** | 忌送粽子（丧事后回礼用）；忌送年糕给丧家 | — |
| **北方** | 忌送鞋（有"送人走"之意）；饺子数量有讲究 | 红包金额较大，攀比现象 |
| **四川** | 绿帽子特别敏感 | — |
| **江浙** | 忌送钟特别忌讳；注意"伞"和"散" | — |
| **港澳** | "空"谐音"凶"，忌送空容器 | — |
| **东北** | 忌分梨吃（"分梨"="分离"） | — |
| **少数民族** | 需了解具体民族禁忌（如回族忌猪肉相关） | — |

## 六、国际送礼禁忌速查

| 国家/地区 | 主要禁忌 | 说明 |
|----------|---------|------|
| **日本** | 忌4/9、忌梳子、忌白花、忌偶数金额 | 收礼不当面拆，回礼文化浓厚 |
| **韩国** | 忌红笔写名字、忌4、忌鞋、忌刀剪 | 双手递接，当面不拆 |
| **俄罗斯** | 忌黄花（丧事/分手）、忌偶数花（丧事）、忌空钱包 | 生日不能提前送礼 |
| **中东/伊斯兰** | 忌猪制品、忌酒、忌左手递物 | 开斋节最重要送礼季 |
| **印度** | 忌牛皮制品（印度教）、忌左手递物、忌黑白包装 | 排灯节最大送礼季 |
| **西方国家** | 忌13、忌现金直接送、忌过于昂贵 | 当面拆礼并感谢 |
| **巴西/拉美** | 忌紫黑包装、忌刀、忌空钱包 | 友谊日(7.20)很重要 |
| **非洲** | 忌左手递物、忌酒（穆斯林）、忌猪制品 | 尊重长辈是核心 |
