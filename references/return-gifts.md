# 回礼指南

## 一、回礼基本原则

| 原则 | 说明 |
|------|------|
| **及时性** | 收到礼物后应在合适时机回礼 |
| **等价或加码** | 回礼价值应≥收到的礼物 |
| **对等性** | 场合对等（婚宴回婚宴，生日回生日） |
| **心意为主** | 不能太敷衍，也不能太刻意 |

## 二、回礼时机

| 场景 | 回礼时机 |
|------|---------|
| 收到生日礼物 | 对方生日时回礼 |
| 收到婚礼红包 | 对方婚礼时回礼 |
| 收到节日礼物 | 下一个节日回礼 |
| 收到拜访礼物 | 回访时带礼物 |
| 收到帮忙感谢礼 | 不必回礼（已两清） |
| 收到领导礼物 | 节日时回礼 |
| 收到客户礼物 | 下次见面时回礼 |

## 三、回礼价值比例

| 场景 | 回礼价值 | 说明 |
|------|---------|------|
| **平辈/朋友** | 等价或略高 (100-120%) | 体现尊重 |
| **长辈给晚辈** | 不必等价 | 晚辈给长辈应加码 (120-150%) |
| **领导给下属** | 下属回礼可略低 (80-100%) | 但不能差太多 |
| **婚宴红包** | 等价或加码 | 至少不能低于对方给的 |
| **客户送礼** | 等价或略高 | 维护关系 |
| **亲戚之间** | 等价为主 | 避免"占便宜"嫌疑 |

## 四、不同场合回礼标准

| 场合 | 回礼标准 |
|------|---------|
| **婚礼** | 红包等价或加码；实物回礼（喜糖、喜饼） |
| **满月/百天** | 红包等价或略高 |
| **乔迁** | 等价礼品或红包 |
| **春节** | 压岁钱等价或略高 |
| **中秋** | 月饼礼盒等价或略高 |
| **探病** | 不必等价，水果/营养品即可 |
| **丧事** | "帛金"等价或略高 |

## 五、回礼话术

| 场景 | 话术 |
|------|------|
| **一般回礼** | "上次您送的礼物太客气了，这个是我的一点心意" |
| **婚礼回礼** | "您来参加婚礼太感谢了，这是喜糖/喜饼，沾沾喜气" |
| **节日回礼** | "上次中秋您送的月饼太好了，这个是我的心意" |
| **加码回礼** | "您上次送的太贵重了，这个虽然不如您的好，但是我的一片心意" |
| **晚辈给长辈** | "您上次给的太客气了，这个是孝敬您的" |
| **领导回礼** | "领导您太客气了，这个不成敬意" |

## 六、回礼禁忌

- **不要立即回礼** - 显得太计较
- **不要回太差的** - 显得不尊重
- **不要回同一件物品** - 显得敷衍
- **不要当面打开对方的礼物再决定回什么**
- **不要说"这个值多少钱"** - 太功利
- **不要在回礼时提"上次你送的XX"** - 显得计较

## 七、特殊情况处理

### 收到不喜欢的礼物

- 仍然要回礼，不能因为不喜欢就不回
- 回礼时不需要回同类型礼物
- 保持礼貌和感谢

### 收到超出预期的贵重礼物

- 回礼应适当加码
- 可以分多次回礼（节日+生日）
- 表达真诚感谢

### 收到现金红包

- 按等价或加码回礼
- 可以回现金，也可以回等价礼物
- 记录金额，避免遗忘

### 收到无法回礼的礼物

- 如收到帮忙后的感谢礼，不必回礼（已两清）
- 如收到长辈的压岁钱，节日时回礼或孝敬
