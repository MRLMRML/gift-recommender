# 国际送礼指南

## 一、西方/欧美通用

### 圣诞节（12 月 25 日）

**最重要的西方送礼季。**

| 对象 | 推荐礼物 | 金额参考 |
|------|---------|---------|
| 家人 | 礼物盒、衣服、电子产品、书籍 | $50-200 |
| 朋友 | 巧克力、酒、蜡烛、围巾 | $20-50 |
| 同事 | Secret Santa 礼物、贺卡 | $10-30 |
| 孩子 | 玩具、书籍、糖果 | $20-100 |

**习俗**：
- 当面拆礼物并表示感谢
- 贺卡很重要
- Secret Santa（秘密圣诞人）在职场非常流行

## 二、日本

### 主要节日

| 日期 | 名称 | 送礼对象 | 推荐礼物 |
|------|------|---------|---------|
| 1.1 | 正月（お正月） | 长辈/亲友 | 御年玉（压岁钱）/年货 |
| 3.3 | 女儿节（ひな祭り） | 女孩 | 人偶/和果子/桃花 |
| 5.5 | 儿童节（こどもの日） | 孩子 | 鲤鱼旗/玩具 |
| **7月** | **お中元** | **上司/客户/恩人** | **礼品盒/食品** |
| 8月中旬 | お盆（盂兰盆节） | 长辈/返乡亲友 | 特产/礼品 |
| **12月** | **お歳暮** | **上司/客户/恩人** | **礼品盒/食品** |

### 重要说明

- **お中元（7月）和お歳暮（12月）是日本最重要的送礼季**，相当于中国的春节送礼
- 日本人极重视包装
- 收到礼物**当面不拆**
- 回礼文化浓厚（"お返し"）
- 收礼时会推辞 2-3 次才收下

### 日本禁忌

| 禁忌 | 原因 |
|------|------|
| 数字 4 | "四"谐音"死"（shi） |
| 数字 9 | "九"谐音"苦"（ku） |
| 梳子 | "梳"（くし/kushi）谐音"苦死" |
| 白色花 | 丧葬用花 |
| 盆栽植物 | 会"生根"，暗示疾病久治不愈 |
| 带尖刺的花 | 不吉利 |
| 领带（给上级） | 有"束缚"之意 |
| 鞋子 | 被踩在脚下，不尊重 |
| 手帕 | 丧葬联想 |
| 偶数金额 | 日本礼金通常是奇数（丧事才用偶数） |

### 韩国禁忌

| 禁忌 | 原因 |
|------|------|
| 红色墨水写名字 | 红色名字代表死亡 |
| 数字 4 | 同中国 |
| 鞋子 | 有"跑掉"之意 |
| 刀/剪刀 | 切断关系 |
| 手帕 | 代表离别和眼泪 |
| 三角形包装 | 不吉利 |

### 韩国习俗

- 双手递接礼物
- 收到礼物**当面不拆**
- 父母节（5.8）比母亲节更重要

## 四、俄罗斯

### 主要节日

| 日期 | 名称 | 送礼对象 | 推荐礼物 |
|------|------|---------|---------|
| 1.1 | 新年（Novy God） | 全部 | **最大送礼季**，礼物/装饰/食品 |
| 1.7 | 东正教圣诞 | 亲友 | 小礼物 |
| **2.23** | **祖国保卫者日** | **男性** | **工具/酒/电子产品** |
| **3.8** | **国际妇女节** | **所有女性** | **鲜花/巧克力/礼物** |
| 5.9 | 胜利日 | 老兵/长辈 | 康乃馨/红玫瑰 |
| 浮动 | 东正教复活节 | 亲友 | 复活节面包/彩蛋 |

### 俄罗斯禁忌

| 禁忌 | 原因 |
|------|------|
| **黄色花** | 关联丧事、分离、不忠 |
| **偶数花束** | 只用于丧事，活人必须奇数 |
| **空钱包/空包** | 送钱包时放一枚硬币 |
| **利器（刀/剪刀）** | 象征切断关系 |
| **手帕** | 关联悲伤和眼泪 |
| **黑/白包装纸** | 关联丧事 |
| **生日前送礼** | 被认为不吉利 |

### 俄罗斯习俗

- 礼物应精美包装
- 主人可能初始推辞，坚持再给
- 当面拆礼物
- 男士拜访女性时应带花
- 巧克力和甜食总是受欢迎
