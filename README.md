## 🎯 项目是什么

Gift Recommender 是一个 AI Skill，帮用户在各种场合做出最合适的送礼决策。

**核心理念**：场合精准、关系得体、预算合理、禁忌不踩。

**工作流程**：

```
用户描述场景 → Agent 读取 SKILL.md → 解析维度 → 加载参考 → 生成方案 → 输出
```

**覆盖维度**：

- 场合（节日/人生事件/商务/求人办事）
- 关系（亲属/领导/客户/朋友/同事）
- 预算（经济/中档/高端/奢侈）
- 地域（全国各省市 + 12个国际区域）
- 时间（30+ 中外节日）
- 禁忌（25+ 文化禁忌速查）

### 交互示例

**节日送礼**：

```
用户：中秋节给领导送什么礼物好？预算500-1000
AI：为您推荐3个方案：
    ⭐ 方案一（首推）：高档月饼礼盒+茶叶
       预算：¥600-800
       理由：中秋标配，体面不出错
    方案二：阳澄湖大闸蟹礼盒
       预算：¥500-800
    方案三：高档水果+红酒
       预算：¥500-700
    
    话术："X总，中秋快乐！一点小心意，祝阖家团圆~"
    
    避雷：不送钟、伞、梨、单数礼
```

**份子钱查询**：

```
用户：朋友结婚，份子钱随多少合适？我在北京
AI：根据关系和城市：
    普通朋友：¥500-800
    好友/闺蜜：¥1000-2000
    吉利数字：666、888、1314
    
    避雷：忌4、忌单数、用新钞
```

**求人办事**：

```
用户：求领导帮忙调动工作，怎么送礼？
AI：核心策略：借节日之名，循序渐进
    
    推荐礼品：
    ⭐ 高档茶叶礼盒（铁观音/大红袍）
    名酒（茅台/五粮液，如果领导喝酒）
    购物卡
    
    金额：¥2000-8000（视关系和难度）
    
    话术："领导，我在现在岗位学了很多，但家里确实有困难，
          想请您帮忙指点指点"
    
    注意：先试探态度，不要贸然送礼
```
